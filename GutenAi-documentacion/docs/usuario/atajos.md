# Lista completa de atajos de teclado de GutenAI

## Archivo
- `Ctrl+O`: Abrir EPUB.
- `Ctrl+Shift+O`: Abrir carpeta proyecto.
- `Ctrl+N`: Nuevo proyecto.
- `Ctrl+S`: Guardar cambios.
- `Ctrl+Shift+E`: Exportar EPUB.
- `Ctrl+Q`: Salir.

## Edición
- `Ctrl+Z`: Deshacer.
- `Ctrl+Y`: Rehacer.
- `Ctrl+X`: Cortar.
- `Ctrl+C`: Copiar.
- `Ctrl+V`: Pegar.
- `Ctrl+A`: Seleccionar todo.

## Búsqueda
- `Ctrl+F`: Buscar en documento.
- `Ctrl+H`: Buscar y reemplazar.
- `Ctrl+Shift+H`: Buscar y reemplazar en todo el libro.
- `F3`: Buscar siguiente.
- `Shift+F3`: Buscar anterior.

## Formato HTML
- `Ctrl+P`: Convertir a párrafo.
- `Ctrl+1`: Encabezado H1.
- `Ctrl+2`: Encabezado H2.
- `Ctrl+3`: Encabezado H3.
- `Ctrl+Shift+Q`: Cita (blockquote).
- `Ctrl+L`: Vincular estilos CSS.
- `Ctrl+Alt+U`: Lista no ordenada (ul).
- `Ctrl+Alt+O`: Lista ordenada (ol).
- `Ctrl+Shift+I`: Elemento de lista (li).

## Navegación
- `F9`: Mostrar/ocultar estructura.
- `F10`: Mostrar/ocultar previsualización.
- `F11`: Previsualización pantalla completa.
- `F5`: Recargar previsualización.
- `Ctrl+G`: Generar navegación (TOC).

## Recursos
- `Ctrl+Shift+N`: Crear nuevo documento.
- `Ctrl+Shift+S`: Crear nuevo CSS.
- `Ctrl+Shift+M`: Importar imagen.
- `Ctrl+Shift+T`: Importar fuente.
- `F2`: Renombrar recurso.
- `Ctrl+Shift+R`: Renombrado en lote.
- `Ctrl+Delete`: Eliminar recurso seleccionado.

## Estadísticas y herramientas
- `F6`: Estadísticas del capítulo actual.
- `Ctrl+F6`: Estadísticas del libro completo.
- `Ctrl+Shift+V`: Validar EPUB.

## IA y corrección
- `Ctrl+Shift+F7`: Corrector IA.

## Ayuda
- `F1`: Mostrar atajos de teclado.
- `Ctrl+Shift+P`: Preferencias.

Total: 42 atajos.
