## Contribución
- Sigue PEP 8, usa type hints y docstrings en español.
- Coloca assets bajo `docs/assets/<sección>` y enlaza con rutas relativas.
- Mantén los cambios enfocados y describe el propósito en los PRs.

## Publicación
Este sitio se gestiona con MkDocs (ver [mkdocs.yml](mkdocs.yml)). Si necesitas construir o publicar la web de documentación, utiliza la guía interna del equipo o abre un issue para coordinar el flujo de publicación.
